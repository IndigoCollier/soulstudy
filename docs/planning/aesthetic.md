# SoulStudy — Aesthetic Guide

## Mood
Imagine walking into a cozy, candlelit study lounge with shea butter in the air, lo-fi beats playing, warm wood and textiles everywhere. **Grounding, nurturing, empowering.**

## Color Palette

| Role | Name | Hex | Usage |
|---|---|---|---|
| Background | Deep Espresso | `#1A1210` | Main app background |
| Surface | Warm Charcoal | `#2A2220` | Cards, sidebars, modals |
| Surface Elevated | Cocoa | `#3A302C` | Hover states, active cards |
| Primary | Terracotta | `#C8674E` | Buttons, active states, links |
| Accent | Warm Amber Gold | `#D4A853` | Highlights, icons, badges |
| Purple Glow | Soft Plum | `#8B6EA4` | Secondary accent, active tabs, gradients |
| Purple Deep | Midnight Plum | `#3D2B50` | Subtle backgrounds, hover overlays |
| Text Primary | Cream | `#F5F0E8` | Headings, body text |
| Text Secondary | Warm Sand | `#B8A99A` | Captions, labels, subtle text |
| Success | Sage Green | `#7A9E7E` | Progress, completed states |
| Border | Soft Umber | `#4A3E38` | Dividers, card borders |

## Typography
- **Headings**: Playfair Display (serif)
- **Body**: Inter / DM Sans (sans-serif)
- **Accents**: Space Mono (monospace) — timers, code-like elements

## Design Elements
- Mudcloth-inspired subtle patterns as section backgrounds (very low opacity)
- Warm-to-purple gradients (terracotta → amber gold, plum → gold)
- Pill-shaped buttons/tabs with rounded borders
- Rounded corners (16px+)
- Gentle amber + purple glow on focus states
- Decorative ✦ sparkle dividers between sections
- 300ms ease-in-out transitions
- Generous whitespace — let the design breathe
- Inspirational quote as hero element on dashboard

## Inspiration
- Soul Whispers app aesthetic (dark warm bg, gold serif text, pill nav, minimal spacing)
